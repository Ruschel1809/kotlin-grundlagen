# Kotlin Grundlagen

Übungen zu Kotlin-Grundlagen aus der Weiterbildung zur
**KI-Entwicklung** (Juni – Dezember 2025).

## Inhalte

| Ordner | Beschreibung |
|---|---|
| `KotlinUebung2101VariablenUndDatentypen` | `val` vs `var`, Datentypen, String-Interpolation |
| `KotlinUebung2201FunktionenUndLambda` | Lambda als Variable, MwSt-Berechnung mit Higher-Order Functions |
| `KotlinUebung2202FunktionenUndLambdas` | Lambda als Funktionsparameter – Logger-Pattern für Temperaturkonvertierung |
| `KotlinUebung2301Kontrollstrukturen` | `if` als Ausdruck, Steuerrechner |
| `KotlinUebung2302Kontrollstrukturen` | `when` als Ausdruck, Null-Safety mit `toDoubleOrNull()`, Rabattregelung |

## Highlights
- Lambda als Funktionsparameter (Higher-Order Functions)
- Idiomatisches Kotlin: `if` und `when` als Ausdrücke
- Null-Safety mit `toDoubleOrNull()` und `toIntOrNull()`

## Technologien
- **Sprache:** Kotlin
- **IDE:** IntelliJ IDEA / Android Studio
